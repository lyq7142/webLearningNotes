响应式布局 Responsive design

实现不同屏幕分辨率的终端上浏览网页的不同展现方式。

自适应布局 Adaptive layout

开发多套页面来适应不同的终端。

流动式布局

用非固定像素定义网页内容（百分比布局）。 通过将盒子的宽度设置成百分比来根据屏幕的宽度伸缩，不受固定像素限制，内容向两侧填充。

弹性布局 flex

display:flex; display:inline-flex; //行内元素 display:-webkit-flex; //safari

flex布局的元素---flex container 容器 所有子元素-------flex item 项目
项目默认水平主轴排列 水平主轴 main axis （main start---main end） 垂直交叉轴 cross axis （cross start---cross end）

容器属性：

flex-direction
（默认）row 左到右 ，column 上到下
flex-wrap
（默认）nowrap 不换行，wrap 换行到下方
flex-flow
前两个简写，(默认)row nowrap
justify-content
项目在主轴对齐方式，(默认)flex-start 左对齐，center 居中，space-betweem 两端对齐
align-items
项目在交叉轴对齐方式，（默认）stretch 全占满，baseline,其它同上
align-content
多根轴线对齐方式，(默认)stretch

项目属性：

order
flex-grow
flex-shrink
